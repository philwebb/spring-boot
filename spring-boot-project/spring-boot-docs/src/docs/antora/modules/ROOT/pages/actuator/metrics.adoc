[[actuator.metrics]]
= Metrics

Spring Boot Actuator provides dependency management and auto-configuration for https://micrometer.io[Micrometer], an application metrics facade that supports {micrometer-docs}[numerous monitoring systems], including:

- xref:actuator/metrics/export.adoc#actuator.metrics.export.appoptics[AppOptics]
- xref:actuator/metrics/export.adoc#actuator.metrics.export.atlas[Atlas]
- xref:actuator/metrics/export.adoc#actuator.metrics.export.datadog[Datadog]
- xref:actuator/metrics/export.adoc#actuator.metrics.export.dynatrace[Dynatrace]
- xref:actuator/metrics/export.adoc#actuator.metrics.export.elastic[Elastic]
- xref:actuator/metrics/export.adoc#actuator.metrics.export.ganglia[Ganglia]
- xref:actuator/metrics/export.adoc#actuator.metrics.export.graphite[Graphite]
- xref:actuator/metrics/export.adoc#actuator.metrics.export.humio[Humio]
- xref:actuator/metrics/export.adoc#actuator.metrics.export.influx[Influx]
- xref:actuator/metrics/export.adoc#actuator.metrics.export.jmx[JMX]
- xref:actuator/metrics/export.adoc#actuator.metrics.export.kairos[KairosDB]
- xref:actuator/metrics/export.adoc#actuator.metrics.export.newrelic[New Relic]
- xref:actuator/metrics/export.adoc#actuator.metrics.export.otlp[OpenTelemetry]
- xref:actuator/metrics/export.adoc#actuator.metrics.export.prometheus[Prometheus]
- xref:actuator/metrics/export.adoc#actuator.metrics.export.signalfx[SignalFx]
- xref:actuator/metrics/export.adoc#actuator.metrics.export.simple[Simple (in-memory)]
- xref:actuator/metrics/export.adoc#actuator.metrics.export.stackdriver[Stackdriver]
- xref:actuator/metrics/export.adoc#actuator.metrics.export.statsd[StatsD]
- xref:actuator/metrics/export.adoc#actuator.metrics.export.wavefront[Wavefront]

TIP: To learn more about Micrometer's capabilities, see its https://micrometer.io/docs[reference documentation], in particular the {micrometer-concepts-docs}[concepts section].
